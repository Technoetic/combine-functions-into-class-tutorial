# Step 030 레이아웃 설계

## 레이아웃

- 상단: 튜토리얼명, 앱 사례 탭, 진행률.
- 본문: 3열 작업대.
- 좌측: 4단계 레일.
- 중앙: 추상 캔버스와 기능 조각, 센터 드롭 존.
- 우측: 현재 관찰 포인트, 지표, 안전 체크.

## 반응형

- 1100px 이상: 3열.
- 760px 이상: 좌측 레일과 중앙/우측을 2열로 재배치.
- 760px 미만: 상단 탭, 단계 레일, 캔버스, 패널 순서의 단일 열.

## 시각 원칙

- 코드 블록 없음.
- 추상 도형은 카드, 링, 흐름선, 칩으로 구성.
- 색은 중립 배경 60%, 작업대 패널 30%, 단일 teal accent 10% 중심.
- 보조 의미는 선, 패턴, 굵기, 라벨로 구분한다.
