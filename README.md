# Combine Functions into Class Tutorial

Martin Fowler의 리팩토링 **Combine Functions into Class**를 초보자가 코드 없이 체감하도록 만든 인터랙티브 웹 튜토리얼입니다.

## 특징

- 배달 앱, 음악 앱, 사진 앱 사례 기반
- 흩어진 기능 조각을 하나의 센터로 모으는 추상 인터랙션
- 학습 화면 코드 스니펫 미노출
- Vanilla JS ES2022 + Vite
- ES6 Class, 생성자 주입, async init 구조

## 실행

```bash
npm install
npm run dev
```

## 검증

```bash
npm run build
npm run test
npm run lint
npx playwright test
```
